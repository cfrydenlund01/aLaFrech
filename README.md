# aLaFrech

Modular Python application that provides a GUI for practicing French vocabulary and
verb conjugations.

## Features

* Flashcard practice with curated vocabulary grouped by topic.
* Present tense verb conjugation drills with streak and accuracy tracking.
* Accent toolbar to quickly insert French characters while typing answers.

## Getting started

The application relies only on the Python standard library. To launch the GUI run:

```bash
python -m french_practice
```

The interface is organized into tabs. Pick a category for flashcards or practice
conjugations for randomly selected verbs. Statistics are displayed at the bottom
of each tab to help track progress.
